Mike's Custom Pane Maker: Instructions

This tool generates responsive HTML banners with smart Velocity logic to personalize student content in TargetConnect based on their tags.

1. Setting Audience Rules (AND logic, sequential check)

This tool creates IF / ELSE IF / ELSE logic. This means rules are checked in order from top to bottom. The first rule that matches the user's tags is the only one they will see.

  • Target Tags: Type comma-separated tags into the field. This rule operates as an "AND" condition. For example, typing Business School, Final Year means the student must have ALL of those tags to see that banner.
  • Case-Insensitivity: Note: All tag checks are Case-insensitive (e.g., 'Business' matches 'business').
  • Limits: While the tool can handle many rules, try limiting to ~15 distinct groups for optimal system performance in TargetConnect.

2. Choosing a Style & Colors (Case-Specific detail)

Five distinct templates are available from the dropdown. Colors will auto-adapt to templates, but you can control them:

  • Solid Background (Solid Color Block): Uses the Primary color for background, and auto-calculates text color.
  • Gradient (Smooth Gradient): Creates a sweep between Primary and Grad 2. A popular choice for a Case 2 modern feel. Tip: Glass-morphism effects apply on pills.
  • Minimal Outlined: Uses Primary color as a thick border, perfect for formal alerts.
  • Notice (Important Notice): Bold styling for urgent deadlines.
  • Professional: Crisp, squared edges for corporate/academic announcements.
  • Custom Image Banner: Paste a link to a graphical banner in the "Image URL" field. The tool will render the image responsively and place buttons underneath. Note: Leave headline blank to only show the image.
  • Automatic Accessibility Checker: This tool Case 2 automatically sets the 'Text' color (White or Black) based on the background color's contrast to guarantee accessibility standards.

3. The Fallback (Default Banner for personalizing)

The **Fallback (Default Banner)** card is the final condition for personalizing content. Students who do not have any tags matching the dynamic rules above will see this standard banner. Leave the headline field blank if you prefer that nothing is shown by default.

4. Buttons & Links (Leap Links & Custom)

You can add up to three buttons Case 4. For each, use the dropdown to select common, internal TargetConnect "Leap Links" like Appointments, Events, or Jobs. Select "Custom URL..." to type in your own internal page or external website link.

5. Getting and Using the Code (Case-Specific output)

Once your rules are set, look at the **👀 Live Preview** to visualize how the logic functions for different student cases. Finally, click the **📋 Copy Velocity Code** button and paste the Case 5 entire output into the HTML Editor pane within a Custom Pane configuration on the TargetConnect staff dashboard.
