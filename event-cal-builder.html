Event Calendar Maker

Generate a branded, interactive calendar that syncs with your TargetConnect Events RSS feed.

Universal Edition 1.0

⚙️ Style & Data

🚨 Required TargetConnect Setup: In TargetConnect RSS settings, ensure 'Show field name' is checked for:
  • Title
  • Event Type
  • Brief Description
  • Location
  • Closest Future Date
  • End Date

💾 Save & Host Instructions

1. Copy the code from the right panel.
2. Open Notepad or TextEdit, paste the code, and save as index.html.
3. This file must be hosted on a secure web server (HTTPS) or GitHub Pages to work.

Security: Your IT team is usually the best point of contact for hosting. If you need help, contact Mike Henson.
